# 🚀 MERN Stack App — Godrej Emerald

This is a **MERN stack** project connected to a shared **Railway MySQL** database so the whole team can work on the same data.

---

## 🧰 Prerequisites
- Node.js (v16+)
- npm or yarn
- Git
- MySQL client (`mysql` command)

---

## 1️⃣ Clone the repo
```bash
git clone <repo_url>
cd <repo_folder>



⚙️ Setting up Environment Variables
Create a .env file in the project root with:

env
PORT=

DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=

JWT_SECRET=supersecretkey
CLIENT_URL=http://localhost:5173

EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_email_app_password



📦 Install dependencies

For backend
cd backend
npm install

For frontend
cd frontend
npm install


▶️ Run the app
Start backend

cd backend
npm start

Start frontend

cd frontend
npm run dev
